# What is Lokey?

Lokey is a framework used for building firmware for a variety of input devices[^1]. The framework includes functionality for keyboards, mice, and MIDI controllers however it is also possible to use it for other kinds of input devices as the framework is extensible.

::: warning
Lokey is still a work in progress. Some of the information here may change over time or may not be completly implemented yet.
:::

[^1]: Input devices are devices that provide data to a host (e.g. key presses, mouse movement, etc.). The host usually is a PC, smartphone, etc.
