# API Documentation

## Core

## Transports

- [lokey_usb](https://docs.rs/lokey_usb) - External USB transport

- [lokey_nrf](https://docs.rs/lokey_nrf) - nRF series microcontrollers

- [lokey_led_array](https://docs.rs/lokey_led_array) - Component for controlling LED arrays
